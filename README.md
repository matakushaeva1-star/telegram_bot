# описание бота
бот пошагово спрашивает у пользователя ФИО, город и телефон.
После ввода всех данных отправляет пользователю готовую анкету

# handlers
обработчики событий от телеграм. 
Хендлер отвечает на конкретное событие 
/tart, /help, /command, текстовое сообщение
callback от inline-кнопки

# Router
Это способ группировать хэндлеры(сгруппированные хендлеры, например по ролям)

# FSM
Finite State Machine - конечный автомат (чтобы он держал диалог)
Используется для пошаговых диалогов (переключение состояний, например состояние ввод имени, ввод фамилии, ввод телефона)
